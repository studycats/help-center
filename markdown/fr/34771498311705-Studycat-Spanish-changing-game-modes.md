### **Comment changer les modes de jeu :**


1\. Accédez à la section parents. Vous pouvez suivre notre guide [ici](https://help.studycat.com/hc/en-us/articles/34518228622105/preview/eyJhbGciOiJIUzI1NiJ9.eyJpZCI6MzQ1MTgyMjg2MjIxMDUsImV4cCI6MTcyMDQxMDgxN30.7hW1u2Miesjcs2XqDuBHBNv7tBPGmmhqN4EJUGeGWJE).


2\. Sélectionnez le mode de jeu


  
![IMG_2670.PNG](https://help.studycat.com/hc/article_attachments/34771475427225)


3\. Choisissez le mode souhaité : mode basique (version précédente) ou mode aventure (nouvelle fonctionnalité améliorée)
