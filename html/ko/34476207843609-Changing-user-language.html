title: 사용자 언어 변경하기 category: faq section: change_lang

Learn English - Studycat

1. 포털 아이콘을 선택하세요.

2. 프로필 아이콘을 선택하세요.

3. '설정'을 선택하세요.

4. 화면에 표시된 숫자를 순서대로 입력하여 나이 확인을 완료하세요. 예시: '912'

5. '언어'를 선택하세요.

6. 원하시는 언어를 선택하세요.

7. 이제 앱이 선택하신 언어로 표시됩니다.

다른 Studycat 앱의 경우

1. Studycat 앱을 열고 메인 메뉴에서 '학부모' 영역으로 이동하세요.

2. 화면에 표시된 숫자를 순서대로 입력하여 나이 확인을 완료하세요. 예시: '924'

3. '설정'을 선택하세요.

4. '언어'를 선택하세요.

5. 원하시는 언어를 선택하세요.

6. 이제 앱이 선택하신 언어로 표시됩니다.
