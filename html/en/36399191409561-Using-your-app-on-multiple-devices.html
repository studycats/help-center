If you purchase on an Apple device, you can access your purchase on an Android device, and vice versa. To do this, you must Sign in with Studycat.

Follow the steps below:

On the device using the Apple ID or Google ID used to make the purchase 1. Install the latest version of the app 2. Navigate to the Settings area from the main menu 3. Tap My subscription 4. Tap Restore purchases 5. Tap Sign in and sign in using the same method as before

On another device 1. Install the latest version of the app 2. From the settings area, tap My subscription and then Sign in using the same method as before

IMPORTANT: When signing in to your Studycat account, make sure you select the same sign-in method you used when creating your account. For example, if you created your account using 'Sign in with Google', use the same method within the app.
