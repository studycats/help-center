Comment changer et créer des profils pour jusqu'à 4 apprenants
Voici comment vous pouvez basculer entre les profils et en ajouter de nouveaux :

Créer de nouveaux profils

  • Appuyez sur l'icône du chat en haut à gauche de l'écran
  • Appuyez sur l'icône « Changer » : repérez le carré avec deux têtes de chat bleues
  • Sélectionnez « + »
  • Choisissez un chat, saisissez le nom et l'âge

Changer de profil

  • Appuyez sur l'icône du chat
  • Appuyez sur l'icône « Changer »
  • Choisissez le profil (chat) souhaité dans la liste

Gérer les profils

  • Modification : Pour modifier un profil, accédez à l'icône « Profils », sélectionnez le profil et choisissez « Modifier »
  • Suppression : il n'est pas possible de supprimer des profils pour le moment.
