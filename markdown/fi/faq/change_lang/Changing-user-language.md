Kun asennat Studycat-sovelluksen, sovelluksen valikot näkyvät laitteesi kielellä.

Voit vaihtaa tämän kielen eri kieleksi kuin laitteessasi.

1. Siirry Asetuksiin
2. Etsi kielivalinta  
3. Valitse uusi kieli

Uutta kieltä käytetään tästä lähtien.
