Learn English - Studycat

1. 點選入口圖示。

2. 點選個人檔案圖示。

3. 點選「設定」

4. 依序輸入螢幕上的數字通過年齡驗證，例如「912」

5. 點選「語言」

6. 選擇您想要的語言

7. 應用程式現在將顯示您所選擇的語言

適用於其他 Studycat 應用程式

1. 開啟您的 Studycat 應用程式，從主選單進入「家長專區」。

2. 依序輸入螢幕上的數字通過年齡驗證，例如「924」

3. 點選「設定」

4. 點選「語言」

5. 選擇您想要的語言

6. 應用程式現在將顯示您所選擇的語言
